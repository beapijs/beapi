---
title: Player
---

The main player object for BeAPI

:::warning

You should never have to construct this class yourself.

:::

## Constructor

```js
new Player(client, player);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| client | [Client](/docs/classes/client) |  ❌   | `null`  | Constructed [Client](/docs/classes/client) instance. |
| player | [IPlayer](https://docs.microsoft.com/en-us/minecraft/creator/scriptapi/mojang-minecraft/player) |  ❌   | `null`  | Minecraft IPlayer Object. |
