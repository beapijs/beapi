Exported method to clear BeAPI intervals.

:::caution

Intervals and timeouts need a custom implementation due to how gametest works. You must use them instead of the default JS implementations.

:::

## Method

```js
clearInterval(interval);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| interval | <Link to="/docs/typedefs/interval">Interval</Link> | âŒ | `null` | Interval object. |

## Returns
void

## Examples

```js
import {
  clearInterval,
  setInterval,
} from 'beapi-core';

// Create a variable with 0
let runtime = 0
// Create an interval the runs every 10 ticks
const interval = setInterval(() => {
  // If variable is more than or equal to 10 kill the interval
  if (runtime >= 10) return clearInterval(interval)
  // Log the current interval number and increment it by 1
  console.warn(`Interval #${runtime++}`)
}, 10)
```
